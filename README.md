# Smart Motor Monitoring System

IoT-based Motor Condition Monitoring System with Artificial Intelligence using Gradient Boosting.

## Features

- Real-time monitoring
- MQTT communication
- Node-RED dashboard
- AI prediction (NORMAL / WARNING / FAULT)
- Failure risk analysis
- Maintenance recommendation

## Hardware

- ESP32
- DHT22
- Buzzer
- Potentiometer (simulation)

## Software

- Node-RED
- MQTT (HiveMQ)
- Python
- Scikit-Learn

## Dataset

Features:
- Temperature
- Humidity
- Vibration
- Current

Classes:
- NORMAL
- WARNING
- FAULT
